소화기 등록 및 점검 애플리케이션

프로젝트 소개
아두이노 하드웨어와 블루투스(BLE) 통신으로 연동하여 주변의 소화기를 등록하고, 소화기의 상태(소리 및 빛)를 원격으로 점검할 수 있는 모바일 애플리케이션입니다.

주요 기능 (Features)

1. 소화기 등록 화면 
- 소화기 이미지를 첨부할 수 있는 칸 제공.
- 소화기 등록 버튼: 정보를 입력하고 등록하면 소화기 목록 화면으로 자동 이동.
- 소화기 목록 버튼: 등록된 소화기 리스트를 확인할 수 있는 화면으로 진입.

2. 소화기 목록 화면
- 진입 제한 로직: 등록된 소화기가 없을 경우, "저장된 소화기가 없습니다"라는 메시지와 함께 목록 진입이 차단됩니다.
- 소화기 리스트: 그동안 저장된 소화기 목록을 한눈에 확인 가능.
- 삭제 기능: 리스트에서 등록된 소화기를 선택하여 삭제 가능.

3. 소화기 상세 및 점검 화면
- 블루투스 연결 버튼: 아두이노 기기와의 무선 연결을 위한 블루투스 연동 기능.
- 하드웨어 점검 버튼: 아두이노와 연동하여 소화기의 `소리`와 `빛`이 정상 작동하는지 점검하는 버튼 제공.
- 추가 사진 첨부: 초기 등록 시 사진을 첨부하지 않은 경우, 상세 화면에서 추가로 사진을 등록할 수 있는 칸 제공.

 기술 스택 
- Frontend: Flutter / Dart
- Hardware: Arduino, Bluetooth (BLE) Module
